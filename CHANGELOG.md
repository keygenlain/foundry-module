# Changelog

All notable changes to the Spell and Steel Foundry VTT system module will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-07

### Added
- Initial release of Spell and Steel system module for Foundry VTT v12+
- **Character Creation Wizard** - Interactive 5-step guided character creation
- Character sheet with full attribute and skill management
- NPC/Monster sheet with customizable tags
- Item system supporting weapons, armor, spells, equipment, and loot
- Dice pool system with exploding dice mechanic
- Success counting (4+ = success)
- Spell management with tier and mana cost tracking
- Focus Token economy
- Combat mechanics (attack rolls, damage calculation)
- Chat message integration for all rolls
- Responsive CSS styling
- Complete documentation in README

### Features
- 23 skills categorized by attribute
- 5 ancestries with special abilities (Dwarf, Ratfolk, Human, Orc, Elf)
- 5 item types (weapon, armor, spell, equipment, loot)
- Monster tags (UNDEAD, VOID, HOLY, DEMONIC, CONSTRUCT)
- 46 spells from core Spell and Steel
- Customizable Success Thresholds
- Status effects (Blinded, Burning, Poisoned, Shaken, Restrained)

### Compatibility
- Foundry VTT v12+
- Tested on v12.305

### Known Issues
- Opposed rolls require manual setup
- Enchantments tracked in descriptions (not automated)
- Combat tracker integration planned for v1.1
